# ClaudeTrack

AI coding cost & session management platform.

## Stack
- Next.js 15 App Router + TypeScript
- Tailwind CSS
- Supabase (Auth + PostgreSQL)
- Lemon Squeezy (payments)
- Resend (email)
- Vercel (deployment)

## Package Manager
- Use bun ONLY (never npm/yarn)

## Commands
- bun dev
- bun run build
- bun run lint
